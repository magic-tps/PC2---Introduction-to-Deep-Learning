# Clasificacion de celulas sanguineas - Semana 6

Proyecto para comparar CNNs entrenadas desde cero, el efecto de Batch Normalization y transfer learning con ResNet-18 sobre un dataset de celulas sanguineas.

## Estructura

```text
dl-semana6-apellidos/
|-- README.md
|-- requirements.txt
|-- data/
|   `-- download_data.sh
|-- notebooks/
|   |-- 01_eda.ipynb
|   |-- 02_lenet_vgg.ipynb
|   `-- 03_transfer.ipynb
|-- src/
|   |-- models.py
|   |-- train.py
|   |-- utils.py
|   `-- make_report.py
|-- results/
|   |-- checkpoints/
|   |-- figures/
|   `-- tables/
`-- informe.pdf
```

## Instalacion

```bash
python -m venv .venv
source .venv/bin/activate  # Windows PowerShell: .venv/Scripts/Activate.ps1
pip install -r requirements.txt
```

## Dataset

El dataset listo para clasificacion esta en `../dataset2-master/dataset2-master`, con la estructura `images/TRAIN` e `images/TEST`. La carpeta `dataset-master` contiene la version original con imagenes, anotaciones y `labels.csv`; para estos experimentos se usa `dataset2-master`.

Para copiarlo al repo:

```bash
bash data/download_data.sh
```

Tambien puedes indicar una ruta manual:

```bash
DATASET_DIR=/ruta/a/dataset bash data/download_data.sh
```

Los scripts y notebooks tambien detectan automaticamente `../dataset2-master/dataset2-master` si `data/dataset` no existe o esta incompleto.

## Ejecucion

Ejecutar notebooks completos:

```bash
python -m nbconvert --to notebook --execute notebooks/01_eda.ipynb --inplace
python -m nbconvert --to notebook --execute notebooks/02_lenet_vgg.ipynb --inplace
python -m nbconvert --to notebook --execute notebooks/03_transfer.ipynb --inplace
```

En los notebooks, deja `LIMIT_SAMPLES = None` para resultados finales.

## Scripts principales

Tarea 1, cuatro modelos desde cero:

```bash
python src/train.py --mode scratch --epochs 10 --batch-size 64 --lr 0.001
```

Tarea 2, comparacion controlada con Batch Normalization y LR alta:

```bash
python src/train.py --mode bn --epochs 10 --batch-size 64 --lr 0.001 --high-lr 0.01
```

Tarea 3, transfer learning con ResNet-18:

```bash
python src/train.py --mode transfer --epochs 5 --batch-size 32 --lr 0.0001
```

Generar PDF:

```bash
python src/make_report.py
```

## Resultados

Las corridas escriben:

- `results/tables/summary.csv`: parametros, tiempo promedio por epoca, accuracy en test y epocas hasta 80% de accuracy de validacion.
- `results/tables/*_history.csv`: perdida y accuracy por epoca.
- `results/figures/*.png`: imagenes usadas por los notebooks y el informe.
- `results/figures/extra_autogenerated/`: curvas y matrices extra guardadas automaticamente por `src/train.py`.
- `results/figures/bn_loss_comparison.png`: comparacion controlada sin/con BN.
- `results/figures/bn_high_lr_loss_comparison.png`: comparacion con learning rate alto.
